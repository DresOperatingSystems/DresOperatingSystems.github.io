Welcome to DresOS

Hello, we are DresOS, an open source organization dedicated to enhancing security and development for Android devices, Linux machines, apps, and bots. Our mission is to provide privacy-focused, secure and innovative tools that empower users in a digital world.

To explore our work, check out the tabs: Learn about our ecosystem in DresEcoVerse, get updates on our upcoming Linux distro, discover our secure communication app in DresSecureComms, and browse our GitHub projects for more open source contributions.

FOSS Apps Section

The FOSS Apps section in the EcoVerse is a curated collection of the best privacy-focused, security-enhancing and quality-of-life apps for Android. For a couple of them we went the extra mile by writing detailed descriptions to help users get the most out of each app.

DresTor

DresTor is a refreshed and maintained fork of an older Tor client from The Guardian Project that had unfortunately fallen out of active development. We updated it and gave it a proper home so it could continue to serve the FOSSverse.

DresGeoSpoofing

DresGeoSpoofing is a fork and improvement of an excellent location spoofing app originally created by Matias and his team. This was a learning exercise for us as we want to begin developing our own original apps and it was our way of saying thank you to Matias for actually inspiring us to work on apps. We left all credits to Matias and his team in the app so others can discover their outstanding open source work.

DresSecureComms

Our first original app: DresSecureComms is a FOSS all-in-one secure communication app by DresOS.

Features:

Encrypted SMS

Messages are AES-encrypted before sending.

VOIP/Normal Calling

Choose between Linphone SIP VOIP or standard calls. Configure SIP in settings. (VOIP is due to have its own full suite coming in future updates)

Secure Contacts

Stored in encrypted database, synced from device locally.

Geo Spoofer

User-configurable mock location (set lat/long in settings; enable in dev options).

App Lock

Biometric/PIN protection.

Full Encryption

Database uses SQLCipher.

SS7 Protection

This feature isn't fully part of this app version but it's due to come in a future update so use VOIP to avoid telecom vulnerabilities for now.

Tracker Detection

Scan files/photos/links/URLs with VirusTotal API (add your API key in settings; scans for viruses/loggers).

IP Spoofing

Basic VPN setup provided; also due its own full suite soon so you can tweak it how you like.

Spoofed Headers

Custom HTTP headers, no cookies.

Metadata Wipe

It only removes geo data for now will have more tags to remove in a future update.

How it works:

All data

Encrypted at rest and in transit where possible.

For encrypted SMS

Share keys with recipients manually (will be adding in an automatic way in a future update).

VOIP

Uses Linphone SDK; register with a SIP provider.

To detect SS7

App prefers secure channels but cannot detect attacks directly (USE VOIP FOR NOW).

Internet permission

System-managed and only has access to the features that require internet VOIP, VirusTotal, etc. (sms, normal phone calls etc stay untouched)

Coming early 2026 with a store on F-Droid.
